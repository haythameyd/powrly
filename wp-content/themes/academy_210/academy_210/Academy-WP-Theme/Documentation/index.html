Theme Options

This theme has built-in options panel, you can find it in Appearance/Theme Options section.

Widgets

Blog Authors

Shows selected blog authors.

Latest Tweets

Shows latest tweets feed.

Shortcodes

  1. Click button in the text editor.
  2. Choose shortcode from the dropdown list.
  3. Fill in all required fields and click "Insert Shortcode".

Button

Styled button. Requires color, URL, target and size parameters.

[button size="small" color="dark" url="#" target="self"]Small Dark Button[/button]

Columns

Content column. Requires column width parameter and content.

[one_half]One Half Column[/one_half]

Courses

Shows selected courses. Requires order, columns, number and category parameters.

[courses number="3" columns="3" category="0"]

Content

Filters course and lesson content, requires type parameter and content.

[content type="public"]Content[/content]

Contact Form

Shows contact form.

[contact_form]

Google Map

Shows Google Map. Requires latitude, longitude and zoom parameters.

[map latitude="10" longitude="10" zoom="10"]

Image

Shows styled linked image. Requires image URL and link parameters.

[image link="URL"]URL[/image]

Plan

Shows pricing plan. Requires plan ID.

[plan id="0"]

Player

Shows media player. Requires comma separated file URLs and file title.

[player url="URL"]File Title[/player]

Posts

Shows selected posts. Requires number, category and order parameters.

[posts number="3" order="date" category="0"]

Section

Styled content section. Requires content and title.

[section title=""]Content[/section]

Slider

Images slider. Requires image URLs and caption parameters.

[slider][slide url="URL"]Caption[/slide][/slider]

Testimonials

Shows testimonials slider. Requires number and order parameters.

[testimonials number="3" order="date"]

Tabs

Requires comma separated tab titles and content for each tab.

[tabs]
[tab title="Title One"]Content One[/tab]
[tab title="Title Two"]Content Two[/tab]
[/tabs]

Toggles

Requires toggle title and content.

[toggles][toggle title="Toggle"]Toggle Content[/toggle][/toggles]

Users

Shows selected users. Requires order and number parameters.

[users number="3" order="date"]
